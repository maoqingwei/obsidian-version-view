# Version View - Obsidian 版本管理插件

> **说明：** 本插件是基于原作者 [joeytoday](https://github.com/joeytoday) 的 [Version View](https://github.com/maoqingwei/obsidian-version-view) 插件重新构建的版本。原仓库缺少 `main.js` 文件，因此使用 AI 重新编写了完整代码，保留了所有原有功能并进行了优化。

一个简单直观的 Obsidian 版本管理插件，支持保存、管理、对比和恢复文件版本。

## 功能特性

- **一键保存版本** - 保存当前文件版本，支持自定义版本名称
- **可视化版本对比** - 并排对比差异，带行号显示，新增内容绿色高亮，删除内容红色高亮
- **一键恢复版本** - 瞬间回退到任意历史版本
- **编辑版本信息** - 随时修改版本名称和描述
- **任意版本对比** - 勾选两个版本进行对比，或与当前文档对比
- **全屏对比模式** - 全屏查看差异，更清晰的阅读体验
- **只显示差异** - 过滤相同内容，只关注变化的部分
- **可配置设置** - 自定义版本存储路径和最大版本数量

## 安装方法

### 方式一：手动安装（推荐）

1. 从 [Releases](https://github.com/maoqingwei/obsidian-version-view/releases) 下载以下文件：
   - `main.js`
   - `styles.css`
   - `manifest.json`

2. 在 Vault 的 `.obsidian/plugins/` 目录下创建 `version-view` 文件夹：
```
你的Vault/
└── .obsidian/
    └── plugins/
        └── version-view/
            ├── main.js
            ├── styles.css
            └── manifest.json
```

3. 将下载的文件复制到 `version-view` 文件夹中
4. 重启 Obsidian
5. 进入 **设置 → 社区插件**，找到 "Version View" 并启用

### 方式二：BRAT 安装

1. 安装 [BRAT](https://github.com/TfTHacker/obsidian42-brat) 插件
2. 添加仓库地址：`https://github.com/maoqingwei/obsidian-version-view`

## 使用方法

1. 打开任意笔记
2. 点击左侧边栏的 **时钟图标** 🕐，或使用命令面板搜索"切换版本视图"
3. 在顶部输入框填写版本名称（可选），点击 **"保存此版本"**
4. 版本列表操作：
   - ✏️ 编辑版本名称和描述
   - 🔍 与当前文档对比
   - ↩️ 恢复到此版本
   - 🗑️ 删除该版本
5. 勾选两个版本前的复选框，点击顶部出现的 **"对比"** 按钮，可对比任意两个历史版本

## 对比视图功能

- **只显示差异** - 勾选后仅显示新增和删除的行
- **全屏模式** - 点击"全屏"按钮全屏查看对比结果
- **行号显示** - 两侧分别显示对应版本的行号

## 设置

进入 **设置 → 社区插件 → Version View**：

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 版本存储文件夹 | `res/versions` | 版本文件保存位置 |
| 最大版本数量 | 50 | 超出后自动删除旧版本 |

## 适用场景

- 📝 **写作场景** - 保存文章的不同阶段：大纲→初稿→修改稿→终稿
- 📚 **学习笔记** - 记录知识的迭代过程，保留重要的理解版本
- 🔬 **研究记录** - 实验数据的多版本管理，思路演进的完整追溯
- 💼 **工作文档** - 方案的多轮迭代，会议纪要的重要版本留存

## 常见问题

**Q：保存版本会占用很多空间吗？**  
A：每个版本保存完整文件副本，建议设置合理的最大版本数量（10-30 个为宜）。

**Q：删除的版本能恢复吗？**  
A：不能，删除是永久性的，请谨慎操作。

**Q：移动端可以用吗？**  
A：支持！该插件同时支持桌面端和移动端。
